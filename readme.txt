This plugin makes it simple to add CalcFusion API to your WordPress

== Description ==
CalcFusion is a computation engine that lets you integrate your Excel calculations into your applications.

== Installation ==
1. Upload `calcfusion-wp` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set your CalcFusion account credentials to the Settings (Admin > Settings > CalcFusion)
